SNT - Web Game 2025



  • Ce jeu a pour objectif de manipuler deux notions essentielles du Web : le langage HTML, dans lequel est écrit le code source des pages Web, et les URL .
  • Pour passer au niveau suivant, il vous faudra trouver un flag (la réponse à une énigme). Pour cela il faut examiner avec minutie tous les éléments de la page pour trouver un indice (titre de la page, textes, URL, images, ... ainsi que parfois le code source).
  • Mis à part dans la première et la dernière salle, il faudra simplement modifier l'URL avec le flag.
  • Vous pouvez bien entendu faire des recherches sur le Web pour résoudre une énigme.
